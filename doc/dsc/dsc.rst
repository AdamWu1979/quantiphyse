Dynamic Susceptibility Contrast (DSC) MRI
=========================================

*Widgets -> DSC-MRI*

.. image:: /screenshots/dsc_widget_menu.png

The DSC-MRI package provides a Bayesian modelling tool for quantification of perfusion and other haemodynamic parameters 
from Dynamic Susceptibility Contrast perfusion MRI of the brain.
